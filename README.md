Tools

### send-statsd.py

#### Prerequisites
- Python 2.7.x
- Python [PIP](https://pip.pypa.io/en/latest/installing.html)
- [statsd](https://pypi.python.org/pypi/statsd) Python module

### Installing statsd

1. Run the following command
     ```bash
     $ pip install statsd
     ```

### Running the script
1. Run the following command
     ```bash
     $ src/main/python/send-statsd.py
     ```
